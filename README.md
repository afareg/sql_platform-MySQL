# sql_manage_platform-MySQL
## 基于django和inception，带权限控制的mysql语句运行系统
### 开发环境：
#### django:1.8.14
#### python:2.7.12
### python依赖组件：
#### django-crontab
#### django-simple-captcha
#### django-sslserver
#### django-ratelimit
#### MySQL-python

### 启动：
#### python manage.py runsslserver 0.0.0.0:8000
